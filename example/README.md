To run the example, you need an input data file. You can download it with either of the following commands:

```sh
curl -O https://raw.githubusercontent.com/ClaudiaBrauer/WALRUS/refs/heads/master/demo/data/PEQ_Hupsel.dat
```

```sh
wget https://raw.githubusercontent.com/ClaudiaBrauer/WALRUS/refs/heads/master/demo/data/PEQ_Hupsel.dat
```

or navigate to the URL and download it in your browser.
